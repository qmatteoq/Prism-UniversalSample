# Prism for Universal Windows apps — samples

Samples that show how to use **Prism** in Universal Windows Platform (UWP) applications,
covering the building blocks of a Prism app: bootstrapping, view-model location,
navigation, commands and dependency injection.

> **Historical sample.** These projects target UWP on Windows 10. For new apps consider
> **WinUI 3 / Windows App SDK** together with a modern MVVM toolkit.

## What's included

| Folder | What it demonstrates |
| --- | --- |
| `Prism-BasicDemo` | A minimal Prism UWP app: bootstrapper, automatic view-model wiring and a first page/view-model. |
| Additional folders | More advanced Prism features such as navigation, commands and services. |

## Prerequisites

- Windows 10
- Visual Studio 2015 / 2017 with the **Universal Windows Platform development** workload
- Windows 10 SDK

## Getting started

1. Clone the repository.
2. Open the `.sln` inside the sample folder you want to explore.
3. Restore NuGet packages and run (F5).
